# uas-mechine-learning-# UAS Machine Learning - Decision Tree

## Deskripsi
Implementasi algoritma Decision Tree untuk klasifikasi Dataset Iris
menggunakan Python dan scikit-learn.

## Dataset
- Iris Dataset (built-in scikit-learn)

## Model
- Decision Tree Classifier
- Criterion: Gini
- Max Depth: 3

## Library
- numpy
- pandas
- matplotlib
- scikit-learn
